<!-- GC Pouches Website -->

<!-- Project Overview -->

GC Pouches is a web-based platform developed as a bachelor's degree project. The purpose of the project is to demonstrate how modern web technologies can be used to create a professional online presence for a small business.

The platform allows visitors to browse products, learn about the company, contact the business, and access information through a responsive and user-friendly interface.

<!-- Technologies Used -->

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL
* XAMPP

<!-- Main Features -->

* Responsive Design
* Product Catalog
* About Page
* Contact Form
* Google Maps Integration
* Age Verification (18+)
* Dynamic Product Loading from MySQL Database

 <!-- Installation -->

1. Install XAMPP
2. Start Apache and MySQL
3. Create the database in phpMyAdmin
4. Import the required tables
5. Place the project folder inside htdocs
6. Open localhost/gc-pouches
